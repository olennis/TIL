### hover?

css 에서의 hover 선택자는 마우스를 대상 요소 위에 올려놨을 때 어떻게 변하는지를 알게 되는 효과이다.

hover 선택자는 링크 뿐만이 아닌 모든 요소에 사용 가능하다.

### how to use

```jsx
/* unvisited link */
a:link {
  color: green;
}

/* visited link */
a:visited {
  color: green;
}

/* mouse over link */
a:hover {
  color: red;
}

/* selected link */
a:active {
  color: yellow;
}
```
